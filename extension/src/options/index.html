Sites surveillés

Confidentialité

Méthodologie

Deux méthodologies embarquées en v0.6.0 : AFNOR SPEC 2314 (référentiel français) et EcoLogits 2026-01 (peer-reviewed JOSS 2025). Le choix se fait depuis la popup. Sources et formules détaillées dans l'app desktop /methodologies.

AFNOR SPEC 2314

Sobr.ia

Référentiel français de mesure de l'empreinte LLM. Coefficient K_DECODE = 25 mJ/token/B recalibré audit B (ADR-0012). Point-estimate sur les P50 des distributions (le Monte-Carlo N=10⁴ reste côté app desktop).

AFNOR SPEC publique, code Sobr.ia MIT

EcoLogits 2026-01

GenAI Impact

Port direct des formules f_E, f_L, n_GPU (Rincé & Banse, JOSS 2025). Marge ≤ 2 % vs Python notebook. Calibré sur le serveur p5.48xlarge (8× H100) avec batch B=64 (vLLM).
