## Logos disponibles

- `logo.svg` — wordmark complet (220 × 64), pour en-têtes app, README, vidéo démo.
- `logo-mark.svg` — glyphe feuille seul (48 × 48), favicon + badge extension.
- `logo-mark-onbg.svg` — glyphe seul sur fond `#0d1117` (48 × 48), badge sombre sur surface claire.

## Icônes traits

Le set **Lucide** est consommé via CDN dans les UI kits :

```
https://unpkg.com/lucide-static@latest/icons/<name>.svg
```

Liste des noms Lucide effectivement utilisés dans les UI kits :
`calculator, library, scale, line-chart, upload, globe, download, file-text, book-open, settings, help-circle, chevron-down, chevron-right, arrow-left, arrow-up-right, search, filter, copy, share-2, x, plus, check, alert-triangle, lock, leaf, info, sliders-horizontal, file-down, history`.

Si vous remplacez par Heroicons / Phosphor / Tabler, la grille (24 × 24, stroke 1.5 px) reste compatible — seul l'import change.
